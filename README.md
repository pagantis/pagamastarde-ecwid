pagamastarde-ecwid
================

Módulo de pago de Paga+Tarde para ecwid

## Instrucciones de Instalación

1. Crea tu cuenta en pagamastarde.com si aún no la tienes [desde aquí](https://bo.pagamastarde.com/)
2. Instala el módulo siguiendo las instrucciones proporcionadas en el manual PDF.

Ten en cuenta que para hacer cobros reales deberás activar tu cuenta de Paga+Tarde.

### Soporte

Si tienes alguna duda o pregunta no tienes más que escribirnos un email a soporte@pagamastarde.com.
